Employee Own Record

This module useful to show only his/her records. Manager can see his/her only employees.

How it works? Click here to watch video

Key features:
  • Employee can see his/her record or profile only
  • Manager can see his/her record as well his/her related employees.

Set related user in employee.

When employee user login, can see only own employee record.

Set employee as a "Manager" to other employee.

When manager user login, manager can see own record and all other related employees.
